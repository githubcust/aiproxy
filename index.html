使用方法

使用以下格式访问 API：

https://your-domain.pages.dev/api/{service}/{path}

示例：

# OpenAI GPT API POST https://your-domain.pages.dev/api/openai/v1/chat/completions
# Claude API POST https://your-domain.pages.dev/api/claude/v1/messages
# Gemini API POST https://your-domain.pages.dev/api/gemini/v1beta/models/gemini-pro:generateContent

注意： 请确保在请求头中包含相应的 API Key 和必要的认证信息。
